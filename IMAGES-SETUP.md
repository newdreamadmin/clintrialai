## Import Process

1. **Download all images** to your computer
2. **Rename according to guide** above
3. **Organize into directory structure** as shown
4. **Compress images**:
   ```bash
   # Mac/Linux
   for file in *.jpg; do
     convert "$file" -quality 85 "compressed_$file"
   done
   ```
5. **Convert to WebP** for better performance:
   ```bash
   cwebp -q 85 image.jpg -o image.webp
   ```
6. **Upload to `/assets/images/` directory**

## Troubleshooting

### Images not loading
- Check file paths are correct
- Verify image files exist in `/assets/images/`
- Check file permissions (read access)
- Clear browser cache (Ctrl+Shift+Del)

### Image quality issues
- Recompress and re-upload
- Use higher quality source
- Verify minimum dimensions met

### Slow page load
- Further compress images
- Reduce image dimensions
- Use lazy loading
- Convert to WebP format
