# ClinTrialAI Website

## Project Overview
ClinTrialAI is a corporate website designed to showcase the capabilities and services of the company in the field of medical AI. The website features a modern, premium, and minimal design, utilizing HTML5, CSS3, Bootstrap 5, and Vanilla JavaScript.

## Project Structure
```
clintrialai-website
├── index.html          # Main entry point of the website
├── pages
│   ├── about.html     # Information about ClinTrialAI
│   ├── services.html  # Overview of services offered
│   └── contact.html   # Contact information and form
├── assets
│   ├── css
│   │   └── styles.css # CSS styles for the website
│   └── js
│       └── main.js    # JavaScript functionality
└── README.md          # Project documentation
```

## Setup Instructions
1. **Clone the Repository**
   ```bash
   git clone <repository-url>
   cd clintrialai-website
   ```

2. **Open the Project**
   Open the `index.html` file in your preferred web browser to view the website.

3. **Dependencies**
   This project uses Bootstrap 5 for styling. Ensure you have an internet connection to load Bootstrap from the CDN.

4. **Customization**
   - Modify the content in the `pages` directory to update information about ClinTrialAI.
   - Update styles in `assets/css/styles.css` to change the appearance of the website.
   - Add any additional JavaScript functionality in `assets/js/main.js`.

## Additional Notes
- Ensure that all assets are correctly linked in the HTML files.
- Test the website on different devices to ensure responsiveness.
- For any issues or feature requests, please refer to the project's issue tracker.
